# Skill Graph Clustering & Learner Segmentation using Unsupervised Learning

This project is an AI-driven Educational Analytics System designed to identify skill gaps, segment learners based on performance and engagement, predict future skill proficiency, and generate personalized learning recommendations.

The solution integrates K-Means Clustering for learner segmentation, Association Rule Mining for discovering skill relationships, and Random Forest Regression for future skill prediction. The system provides actionable insights through advanced visualizations, recommendation dashboards, and deployment-ready architecture.

## Key Features

* Skill Gap Analysis
* Learner Segmentation using K-Means Clustering
* Association Rule Mining with Apriori Algorithm
* Future Skill Prediction using Random Forest Regression
* Personalized Learning Recommendations
* Interactive Visualizations and Dashboards
* Enterprise-Ready Deployment Architecture
* Scalable Educational Analytics Framework
